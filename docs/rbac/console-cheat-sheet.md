# Console Click-Path Cheat Sheet

- Enable Auth providers:
  - Firebase Console → Build → Authentication → Sign-in method
- Add/see users (Auth):
  - Firebase Console → Build → Authentication → Users
- Get Web app config:
  - Firebase Console → Project settings (gear) → General → Your apps → Web
- Create Firestore collections:
  - Firebase Console → Build → Firestore Database → Start collection
- Service accounts (GCP):
  - Google Cloud Console → IAM & Admin → Service Accounts
