# Glossary

- GCP (Google Cloud Platform): Infrastructure platform for your project
- Firebase: Dev platform (Auth, Firestore, etc.) integrated with GCP
- Project: Container for resources (Auth, Firestore) identified by Project ID
- UID: Unique ID for an Auth user (used for users/{uid} doc)
- Firestore: NoSQL DB for users, invites, registrations, etc.
- Provider: The sign-in method (Google, Email/Password)
